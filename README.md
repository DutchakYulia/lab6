# Лабораторна робота №6

Тема: написання тестів для користувацьких сценаріїв.

## Обраний ресурс

Для виконання роботи створено локальний навчальний ресурс `app/index.html` - невеликий інтернет-магазин з реєстрацією, каталогом, кошиком і оформленням замовлення.

## Реалізовані сценарії

1. Реєстрація нового користувача: заповнення форми, створення акаунта, перевірка повідомлення про успіх і стану сесії.
2. Пошук товару та додавання до кошика: пошук за назвою, перевірка результатів, вибір параметрів товару, додавання до кошика і перевірка суми.
3. Повний E2E-шлях покупки: реєстрація, пошук товару, додавання до кошика, введення адреси, вибір способу оплати та підтвердження замовлення.
